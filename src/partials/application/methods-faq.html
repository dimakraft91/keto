Questions fréquemment posées:

  • J'AI OUBLIÉ DE PRENDRE LES GÉLULES À TEMPS. DOIS-JE PRENDRE UNE DOUBLE DOSE LA PROCHAINE FOIS?

    Si vous avez oublié de prendre les gélules, NE prenez PAS une double dose. Continuez à utiliser le produit comme indiqué.

  • QUEL EST LE NOMBRE MAXIMAL DE CAPSULES QUE JE PEUX PRENDRE PAR JOUR?

    Il est recommandé de prendre 4 gélules par jour.

  • JE SUIS ENCEINTE. PUIS-JE PRENDRE LES CAPSULES?

    Nous vous déconseillons de les prendre si vous êtes enceinte ou si vous venez de devenir maman.

  • À QUI CONVIENT LES CAPSULES?

    Les gélules conviennent aussi bien aux femmes qu'aux hommes de plus de 18 ans.
